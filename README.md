# AI Image Caption Generator

An AI-powered web application that generates captions for uploaded images using Python, Flask, and the BLIP model from Hugging Face.

## Features
- Upload Image
- AI Generated Caption
- Modern UI
- Fast Image Processing

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Hugging Face Transformers
- Pillow
